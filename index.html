About

I'm an independent game developer and 3D artist, currently based in Ribeirão Preto, BR. I'm passionate about creating immersive gaming experiences and bringing imaginative worlds to life through code and art.

My journey in game development started with a fascination for how games work under the hood. This curiosity led me to explore both the technical and artistic sides of game creation, from programming gameplay mechanics to crafting detailed 3D models.

I actively participate in game jams to challenge myself, learn new techniques, and collaborate with other creators. Each project is an opportunity to experiment with different genres, mechanics, and art styles.

When I'm not coding or modeling, I'm exploring new game dev tools, studying game design principles, and working on personal projects that push my creative boundaries.
